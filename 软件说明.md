# PQE 数据分析平台软件说明

## 软件定位

PQE 数据分析平台用于自动读取 CPK 报告和 FAI 报告，完成尺寸数据解析、统计计算、低 CPK 项识别、建议公差计算、Worst Cavity 分析、图表展示和 Excel 报告导出。

## 核心功能

1. 自动识别并解析 CPK / FAI Excel 文件。
2. 重新计算 `mean`、`stddev`、`cp`、`cpu`、`cpl`、`cpk`、`yield`。
3. 按目标 CPK 识别低风险尺寸。
4. 基于目标 CPK 反推建议公差，并计算建议公差下的 `proposed_cpk`。
5. 按 `spc_no` 计算 Worst Cavity。
6. 从文件名 `()` 中提取筛选字段，例如 `Re-striking`。
7. 在用户界面中提供筛选导出、象限图、矩阵散点图、折线图和完整报告导出。
8. 支持通过 PyInstaller 在 macOS 上打包成用户可双击运行的 `.app`。

## 主要文件

- `pqe_phase1_mvp.py`：核心解析、计算和 Excel 导出逻辑。
- `pqe_phase1_ui.py`：Streamlit 用户界面。
- `open_pqe_dashboard.sh`：Linux 桌面启动脚本。
- `PQE Dashboard.desktop`：Linux 桌面图标。
- `mac_package/`：macOS 打包配置和说明。
- `.github/workflows/build-macos-app.yml`：GitHub Actions 自动构建 macOS App。

## 输出报告

完整 Excel 报告包含：

- `Summary`
- `CPK_Summary`
- `CPK_LowRisk`
- `Tolerance_Proposal`
- `FAI_OK_NG`
- `SPC_OK_NG`
- `Worst_Cavity`
- `Raw_Normalized_CPK`
