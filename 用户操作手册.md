# PQE 数据分析平台用户说明手册

## 1. 手册适用对象

本手册面向不熟悉代码和终端命令的普通用户，例如：

- 质量工程师
- PQE 工程师
- SQE 工程师
- 制程工程师
- 项目质量人员

用户只需要会打开软件、选择文件、查看图表和导出结果。

## 2. 软件能做什么

PQE 数据分析平台可以帮助用户：

1. 自动读取 CPK 报告和 FAI 报告。
2. 自动计算 CPK、均值、标准差、良率等数据。
3. 找出低于目标 CPK 的尺寸。
4. 计算建议公差。
5. 找出每个 SPC 编号下最差的 cavity。
6. 按文件、cavity、SPC、FAI 筛选数据。
7. 查看象限图、矩阵散点图和折线图。
8. 导出 Excel 分析报告。

## 3. 打开软件

### 3.1 Linux 桌面

在桌面上双击：

`PQE 数据分析平台`

软件会自动打开浏览器页面。

### 3.2 Mac 桌面

双击：

`PQE Dashboard.app`

如果系统提示“无法验证开发者”：

1. 右键点击 `PQE Dashboard.app`。
2. 选择“打开”。
3. 在弹窗中再次点击“打开”。

## 4. 准备 Excel 文件

请准备两类文件：

1. CPK Report 文件
2. FAI Report 文件

建议文件名包含：

- `CPK`
- `FAI`

例如：

- `xxx_CPK_Report_20260525.xlsx`
- `xxx_FAI_Report_20260523.xlsm`

如果文件名中有括号，例如 `(Re-striking)`，软件会自动提取为筛选字段。

## 5. 读取数据

打开软件后，左侧是数据来源区域。

### 5.1 自动读取工作目录文件

适合文件已经放在软件目录中的情况。

操作步骤：

1. 保持读取方式为 `工作目录自动读取`。
2. 点击 `解析数据`。
3. 等待页面加载完成。

软件会自动寻找文件名中包含 `CPK` 和 `FAI` 的 Excel 文件。

### 5.2 上传文件读取

适合用户手动选择文件的情况。

操作步骤：

1. 在左侧选择 `上传文件`。
2. 上传 CPK Excel 文件。
3. 上传 FAI Excel 文件。
4. 点击 `解析上传文件`。
5. 等待页面加载完成。

## 6. 首页指标说明

页面顶部显示 8 个指标。

### 6.1 总记录数

CPK 记录和 FAI 记录的总数量。

### 6.2 CPK记录数

从 CPK 文件中解析出的尺寸记录数量。

### 6.3 FAI记录数

从 FAI 文件中解析出的尺寸记录数量。

### 6.4 低于目标CPK数

CPK 小于目标值的记录数量。

默认目标值是 `1.33`。

### 6.5 FAI编号尺寸OK数

FAI 尺寸项目中判定为 OK 的数量。

### 6.6 FAI编号尺寸NG数

FAI 尺寸项目中判定为 NG 的数量。

### 6.7 SPC编号OK个数

按 cavity 和 SPC 编号聚合后，整体为 OK 的 SPC 数量。

### 6.8 SPC编号NG个数

按 cavity 和 SPC 编号聚合后，存在 NG 项的 SPC 数量。

## 7. 筛选导出

进入 `筛选导出` 页面。

可按以下字段筛选：

- `source_file`
- `文件名括号字段`
- `cavity`
- `report_type`

筛选完成后，可以：

- 下载 CSV
- 下载 Excel

适合导出某个 cavity 或某个项目阶段的数据。

## 8. 象限图 / 分布图

进入 `象限图` 页面。

该页面参考 sample 图的展示方式，用于查看指定 SPC 编号下，不同 FAI 编号的 `mean` 分布。

操作步骤：

1. 选择 `source_file`，可不选。
2. 选择 `文件名括号字段`，可不选。
3. 选择一个 `spc_no`。
4. 可选择部分 `fai_no`。
5. 如果需要按 cavity 分开查看，勾选 `增加 cavity 维度`。

如果勾选 `增加 cavity 维度`：

- 有 8 个 cavity，就会显示 8 张图。
- 每张图对应一个 cavity。

图中参考线含义：

- `Proposed USL`：建议上限
- `Drawing USL`：图纸上限
- `Nominal`：名义值
- `Drawing LSL`：图纸下限
- `Proposed LSL`：建议下限

## 9. 矩阵散点图

进入 `矩阵散点图` 页面。

该页面用于从 `cavity`、`spc_no`、`fai_no` 中选择任意两个字段进行交叉分析。

颜色表示：

`mean` 平均值

点大小表示：

该组合下的数据数量

适合快速查看不同 cavity、SPC、FAI 之间的差异。

## 10. 折线图

进入 `折线图` 页面。

用户可以从以下字段中固定任意两个：

- `cavity`
- `spc_no`
- `fai_no`

软件会使用剩余字段作为 X 轴，显示 `mean` 的变化趋势。

示例：

如果固定：

- `cavity`
- `spc_no`

则折线图会按 `fai_no` 显示 `mean` 变化。

## 11. Worst Cavity

进入 `Worst Cavity` 页面。

该页面显示每个 `spc_no` 下 CPK 最低的 cavity。

主要字段说明：

- `spc_no`：SPC 编号
- `worst_sheet_kind`：来源类型，例如 CPK 或 Raw data
- `worst_fai_no`：最差记录对应的 FAI 编号
- `worst_cavity`：最差 cavity
- `worst_cpk`：最低 CPK
- `worst_mean`：该记录的均值
- `worst_stddev`：该记录的标准差
- `sample_count`：样本数量

## 12. 完整报告导出

进入 `完整报告导出` 页面。

点击：

`下载完整 Excel 报告`

软件会导出完整分析结果。

Excel 中包含：

- `Summary`
- `CPK_Summary`
- `CPK_LowRisk`
- `Tolerance_Proposal`
- `FAI_OK_NG`
- `SPC_OK_NG`
- `Worst_Cavity`
- `Raw_Normalized_CPK`

## 13. 建议公差说明

建议公差基于当前样本的均值和标准差计算。

它的目的是估算：

如果要达到目标 CPK，例如 `1.33`，规格边界需要放宽到什么位置。

注意：

建议公差只是工程分析参考，不能直接替代正式图纸变更流程。

## 14. 常见问题

### 14.1 软件打不开怎么办？

请确认：

1. 文件没有被删除或移动。
2. 浏览器是否正常安装。
3. 如果是 Mac，是否允许打开未知开发者 App。

### 14.2 解析失败怎么办？

请确认：

1. CPK 和 FAI 文件是否正确。
2. 文件扩展名是否为 `.xlsx` 或 `.xlsm`。
3. Excel 文件是否损坏。
4. 文件是否正在被 Excel 或 WPS 占用。

### 14.3 为什么没有图？

可能原因：

1. 当前筛选条件下没有数据。
2. `mean` 字段为空。
3. 选择的 `spc_no` 没有对应 FAI 数据。

### 14.4 为什么建议公差不能直接作为正式公差？

因为建议公差只基于当前数据统计计算，没有经过设计、公差链、客户规范和工程评审。

正式公差更改应经过内部和客户批准流程。

## 15. 推荐使用流程

1. 打开软件。
2. 读取 CPK 和 FAI 文件。
3. 查看顶部指标。
4. 查看 `CPK_LowRisk` 和建议公差。
5. 查看 `Worst Cavity`。
6. 使用象限图检查指定 SPC 下不同 FAI 和 cavity 的分布。
7. 导出完整 Excel 报告。
8. 将结果用于工程评审。
